# Gradient Descent Explorations

This repository contains a collection of Jupyter Notebooks focusing on various aspects of gradient descent, optimization, and automatic differentiation using PyTorch and manual implementations.

## Files Overview

- **`Autograd vs Analyticsl.ipynb`**
  Compares PyTorch's automatic differentiation (Autograd) with analytical (mathematical) gradient computations.

- **`complete_enumeration_local_mac_windows.ipynb`**
  Demonstrates complete enumeration (brute-force search) as a baseline method for finding optimal model parameters without using gradient-based optimization.

- **`manual_gradient_descent_normalization_local_mac_windows.ipynb`**
  Shows a manual implementation of gradient descent from scratch, highlighting the importance and application of data normalization for stable training.

- **`pytorch_autograd_local_mac_windows.ipynb`**
  A focused look at leveraging PyTorch's Autograd engine to automatically compute gradients instead of writing the derivative formulas manually.

- **`pytorch_optimizer_local_mac_windows.ipynb`**
  Explores the use of PyTorch's built-in optimization algorithms (like SGD or Adam) to simplify and improve the parameter update step in training loops.

- **`quadratic_gradient_descent_local_mac_windows.ipynb`**
  Illustrates the core concepts of gradient descent applied specifically to a simple quadratic function or model framework.

- **`temperature_gradient_descent_local_mac_windows.ipynb`**
  Applies gradient descent techniques to model and predict a temperature-related dataset, providing a practical, real-world context for learning the algorithms.
